# Rahul230892 Tap

## How do I install these formulae?

`brew install rahul230892/tap/<formula>`

Or `brew tap rahul230892/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rahul230892/tap"
brew "<formula>"
```
